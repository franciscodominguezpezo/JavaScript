🏨 Control de Grupos - Hotel Management System
Una aplicación web ligera y eficiente diseñada para que el personal de recepción y reservas pueda gestionar, filtrar e imprimir reportes operativos de grupos de forma sencilla.

English version below

🇪🇸 Versión en Español
✨ Características
CRUD Completo: Crea, lee, actualiza y elimina registros de grupos.

Persistencia Local: Los datos se guardan en el navegador mediante localStorage (no requiere base de datos externa).

Filtro Inteligente: Sistema de filtrado por rangos de fechas para reportes específicos.

Modo de Impresión Optimizado: Diseño especial para impresión en formato horizontal (A4 landscape) que oculta formularios y botones innecesarios.

Interfaz Responsiva: Construido con Bootstrap 5 para una visualización correcta en tablets y PCs.

Validaciones: Control de fechas para evitar que la salida sea anterior a la entrada.

🛠️ Tecnologías utilizadas
HTML5 / CSS3: Estructura y estilos personalizados (incluyendo @media print).

JavaScript (Vanilla): Lógica del negocio y manejo de datos.

Bootstrap 5: Framework de diseño y componentes.

SweetAlert2: Alertas visuales interactivas y elegantes.

Bootstrap Icons: Iconografía clara para la interfaz.

🚀 Instalación y Uso
Clona este repositorio o descarga los archivos.

Asegúrate de tener los archivos index.html, style.css y script.js en la misma carpeta.

Abre index.html en cualquier navegador moderno.

🇺🇸 English Version
✨ Features
Full CRUD: Create, Read, Update, and Delete group records.

Local Persistence: Data is stored in the browser using localStorage (no external database required).

Smart Filtering: Filter system by date ranges for specific operational reports.

Optimized Print Mode: Specialized design for landscape printing (A4) that automatically hides forms and UI buttons.

Responsive Interface: Built with Bootstrap 5 for seamless use on tablets and desktops.

Validations: Date control to prevent checkout dates before check-in.

🛠️ Tech Stack
HTML5 / CSS3: Structure and custom styling (including @media print rules).

JavaScript (Vanilla): Business logic and data management.

Bootstrap 5: UI components and grid system.

SweetAlert2: Beautiful and interactive popup alerts.

Bootstrap Icons: Modern iconography for better UX.

🚀 Quick Start
Clone this repository or download the source files.

Ensure index.html, style.css, and script.js are in the same directory.

Open index.html in any modern web browser.
