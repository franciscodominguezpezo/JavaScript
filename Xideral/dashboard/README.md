# 🏨 Hotel Reception Productivity Hub

> Sistema inteligente de gestión de tiempo y tareas diseñado para Jefes de Recepción.
> _Intelligent time and task management system designed for Front Office Managers._

---

## 🇪🇸 Descripción (Español)

Este proyecto es una herramienta web de alta eficiencia creada para optimizar la carga de trabajo en el departamento de recepción de un hotel. A diferencia de un gestor de tareas convencional, esta aplicación permite medir el **tiempo real** invertido en cada operación, ayudando a identificar cuellos de botella y periodos de alta demanda administrativa.

### ✨ Características Principales

- **Temporizador en tiempo real:** Control de un solo clic para medir la duración de cada tarea.
- **Gestión de Categorías Dinámicas:** Capacidad de crear nuevas categorías al vuelo (ej: "Auditoría", "Gestión de VIPs") con colores generados automáticamente.
- **Diseño Estilo ClickUp:** Interfaz limpia y moderna construida con Bootstrap 5, optimizada para la multitarea.
- **Persistencia de Datos:** Uso de `localStorage` para asegurar que los datos no se pierdan al cerrar el navegador.
- **Lógica de Tarea Única:** Pausa automática de tareas activas al iniciar una nueva para reflejar la realidad operativa.

### 🛠️ Tecnologías Utilizadas

- **HTML5 & CSS3** (Custom Properties)
- **Bootstrap 5** (Layout y Modales)
- **JavaScript Vanilla** (Lógica de negocio y manipulación del DOM)
- **Google Material Icons** (Interfaz visual)

---

## 🇺🇸 Description (English)

This project is a high-efficiency web tool created to optimize the workload in a hotel's front office department. Unlike conventional task managers, this application measures the **real-time** spent on each operation, helping to identify bottlenecks and periods of high administrative demand.

### ✨ Key Features

- **Real-time Timer:** One-click control to measure the duration of each task.
- **Dynamic Category Management:** Ability to create new categories on the fly (e.g., "Auditing", "VIP Management") with automatically generated colors.
- **ClickUp-Style Design:** Clean and modern interface built with Bootstrap 5, optimized for multitasking.
- **Data Persistence:** Uses `localStorage` to ensure data is not lost when the browser is closed.
- **Single-Task Logic:** Automatic pausing of active tasks when starting a new one to reflect operational reality.

### 🛠️ Tech Stack

- **HTML5 & CSS3** (Custom Properties)
- **Bootstrap 5** (Layout & Modals)
- **Vanilla JavaScript** (Business logic & DOM manipulation)
- **Google Material Icons** (Visual UI)
