# 🏢 Control de Grupos / Group Management

### Suites Viena Plaza de España

---

## 🇪🇸 Versión en Español

Una aplicación web ligera y eficiente para la gestión operativa de grupos hoteleros, reservas de salas y servicios de catering. Diseñada para facilitar el seguimiento diario y la generación de reportes impresos.

### 🚀 Características Principales

- **Gestión CRUD Completa:** Registra, edita y elimina grupos de forma intuitiva.
- **Persistencia Local:** Los datos se guardan en el `localStorage` del navegador (no requiere base de datos externa).
- **Sistema de Filtros:** Filtrado inteligente por rangos de fechas.
- **Identificación Visual:** Cada día de la semana tiene un color pastel asignado para una rápida lectura visual.
- **Modo de Impresión Optimizado:** Formato especial `A4` que oculta menús y resalta observaciones para reportes físicos.

### 🛠️ Tecnologías

- **Frontend:** HTML5, CSS3 (Custom Styles), JavaScript (Vanilla).
- **Frameworks:** Bootstrap 5.3.
- **Librerías:** SweetAlert2 (Alertas), Bootstrap Icons.

### 📖 Instrucciones de Uso

1.  **Añadir:** Rellena el formulario de la izquierda y pulsa "Guardar Grupo".
2.  **Editar:** Haz clic en el icono del lápiz (<i class="bi bi-pencil"></i>) para cargar los datos en el formulario.
3.  **Filtrar:** Usa la barra superior para acotar por fechas de entrada.
4.  **Imprimir:** Pulsa "Imprimir Reporte" para obtener una versión limpia en papel.

## 🇺🇸 English Version

A lightweight and efficient web application for the operational management of hotel groups, meeting room bookings, and catering services. Designed to streamline daily tracking and printed report generation.

### 🚀 Key Features

- **Full CRUD Management:** Intuitively create, edit, and delete group entries.
- **Local Persistence:** Data is stored in the browser's `localStorage` (no external database required).
- **Filtering System:** Smart filtering by date ranges for specific search queries.
- **Visual Identification:** Each day of the week is assigned a pastel color for quick visual recognition.
- **Print-Ready Mode:** Special `A4` format that hides UI elements and emphasizes notes for physical reports.

### 🛠️ Technologies

- **Frontend:** HTML5, CSS3 (Custom Styles), JavaScript (Vanilla).
- **Frameworks:** Bootstrap 5.3.
- **Libraries:** SweetAlert2 (Alerts), Bootstrap Icons.

### 📖 How to Use

1.  **Add:** Fill out the form on the left and click "Guardar Grupo" (Save Group).
2.  **Edit:** Click the pencil icon (<i class="bi bi-pencil"></i>) to load data back into the form.
3.  **Filter:** Use the top bar to narrow down the list by arrival dates.
4.  **Print:** Click "Imprimir Reporte" (Print Report) to generate a clean paper-friendly version.
